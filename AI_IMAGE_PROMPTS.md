3. Paste each prompt
4. Generate images
5. Download and rename files

### Option 2: Use DALL-E via ChatGPT Plus

2. Paste prompts one by one
3. Generate and download
4. Rename files accordingly

### Option 3: Use Midjourney (Best Quality)
1. Join Midjourney Discord
2. Use `/imagine` command with prompts
3. Upscale favorite versions
4. Download and rename

## Pro Tips for Better Results

### Add These Modifiers:
- `--ar 1:1` (Midjourney) for square images
- `--style raw` for more photorealistic
- `--quality 2` for higher detail

### Refine Prompts:
- Add "Canon EOS R5, 50mm lens" for more realism
- Add "e-commerce product photo" for cleaner background
- Add specific colors: "turquoise #4ECDC4 and purple #BB8FCE"

### ThrillWorld Brand Colors:
- Primary: Turquoise (#4ECDC4)
- Secondary: Purple (#BB8FCE)
- Accent: Green (#38EF7D)

## After Generating Images

1. **Resize:** Make all images 800x800px
2. **Compress:** Use TinyPNG.com to reduce file size
3. **Save:** Place in `frontend/public/merchandise/` folder
4. **Test:** Refresh your shop page
